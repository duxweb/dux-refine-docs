# Directory Explanation

Here is an explanation of the directory structure created after using the scaffold:

```sh
.
├── mock               # Directory for mock data
├── public             # Directory for public static resources
├── src                # Main directory
│   ├── config         # Configuration files
│   │   ├── app.ts     # Module index generated by Vite plugin
│   │   └── index.ts   # Basic project configuration
│   ├── pages          # Page files
│   │   └── admin      # Example admin module
│   │       ├── article  # Article page
│   │       ├── category # Category page
│   │       ├── home     # Dashboard page
│   │       ├── role     # Role page
│   │       ├── user     # User page
│   │       ├── setting  # Settings page
│   │       ├── locales  # Multilingual configuration
│   │       ├── config   # Configuration data
│   │       └── index.tsx  # Module configuration
│   ├── static          # Static resource files
│   └── index.tsx       # Main entry
├── .eslintrc.cjs       # ESLint configuration
├── vite.config.ts      # Vite configuration file
└── uno.config.js       # Unocss configuration file
```
